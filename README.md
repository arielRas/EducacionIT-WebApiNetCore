# 📚 EducacionIT-WebApiNetCore

## 🏆 Proyecto de Capacitación en Web API con .NET 8

Este repositorio contiene el código fuente desarrollado durante la capacitación en **Web API con .NET** en la plataforma **Educación IT**. El objetivo principal de este curso es aprender los fundamentos de las API en .NET y su aplicación en un proyecto real.

## 📌 Descripción del Proyecto

Este proyecto implementa una **API para gestionar el inventario de una tienda de libros**, permitiendo realizar operaciones CRUD sobre los recursos de la aplicación, como libros, autores, etc.

Se aplican **buenas prácticas** de desarrollo, incluyendo seguridad, autenticación con **JWT e Identity**, documentación con **Swagger**, y conexión con **SQL Server** como base de datos.

## 🚀 Características

✅ Implementación de **CRUD** para la gestión de libros.  
✅ Autenticación y autorización con **JWT e Identity**.  
✅ Base de datos **SQL Server** con Entity Framework.  
✅ Documentación interactiva con **Swagger**.  
✅ Uso de **buenas prácticas** en la arquitectura y el código.
